Introduction

This project is trying to create a lightweight version of the CodeHS javascript graphics library.

Installation

git clone https://gitlab.com/pythonKiwi123/codehs_lib.git
cd codehs_lib/

# install dependencies
yarn
# or
npm install

# compile the typescript
yarn build
# or
npm run build

(the HTML script.module does not work from file:// so you have to open the file in localhost)
